Customize your Bertha.ai plugin with a choice of 25 robot icons, 7 preloader images, or the option to add your own of each. Further features include a name generator and enhanced branding capabilities (pro version).

Bertha.ai is a next-generation AI content plugin for writing high-converting blog posts, landing pages, product pages, and more. 

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress(This will add new Sub Menu in admin panel's side menu name Rebrand` under Bertha AI menu).
3. Go to Bertha AI > Rebrand

-Yes, we have a lite version of Rebrand Bertha.ai plugin with limited features. If you want to take advantage of all the premium features you'll need to purchase a Pro license.

2. Can you use the Rebrand Bertha.ai plugin on the free version of Bertha.ai?
-Yes, it will work with both free version and pro version.

3. Does it work with any WordPress theme? Yes, it will work with any standard WordPress theme.

4. Does it work with any WordPress page builder? Yes, it will work with any WordPress page builder.

5. How many websites can I connect Rebrand Bertha.ai to? 
-You can connect the lite version to as many websites you would like. If you want to use the Pro features, you'll have to select the plan that fits your needs best.
